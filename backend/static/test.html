How it works

This page demonstrates the embeddable AI chat widget. Look for the chat bubble in the bottom-right corner!

The widget connects to your SmartChat backend via WebSocket for real-time streaming responses.

Integration

To embed this widget on any site, just add:

<script src="http://localhost:8000/widget.js" 
        data-api-key="test-key-123"></script>
